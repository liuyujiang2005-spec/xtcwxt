# 财务系统操作指南

## 系统入口

登录系统后进入主界面，左侧导航栏各模块如下。

---

## 1. 仪表盘

默认首页，展示整体经营数据汇总（账单总额、已收、待收、支出等）。

---

## 2. 拼柜管理

### 上传拼柜数据
1. 点击"拼柜管理" → "上传拼柜"
2. 选择 Excel 文件上传，系统会自动识别数据
3. 也可点击"手动录入"，逐条填写：唛头号、品名、尺寸、体积、运单号等
4. 提交后进入"待验证"状态

### 审核拼柜
1. 点击某条批次进入详情
2. 可对每行数据做标记（验证通过/有问题）
3. 审核完成后将批次状态改为"已发布"，数据生效

### 拼柜数据
- 按批次查看所有拼柜明细
- 可按运单号、唛头筛选
- 支持修改单条明细

---

## 3. 装柜管理

操作流程同拼柜：上传 → 验证 → 发布。装柜与拼柜的区别是装柜为一整柜的货物，不按体积分摊。

### 手动录入装柜
1. 进入批次详情 → "手动录入"
2. 逐行填写唛头、品名、单价、总体积等
3. 支持"加一行""删一行"，提交后导入

---

## 4. 唛头管理

- 查看所有唛头列表，按月份、客户筛选
- 编辑唛头信息（关联客户、备注）
- 点击唛头可查看该唛头下的所有明细

---

## 5. 客户管理

### 客户信息
- 查看/新建/编辑客户
- 设置**价格矩阵**：海运普货、海运商检、海运敏感货、陆运普货、陆运商检、陆运敏感货，6 个单价
- 启停**低消规则**（海运 0.5 方、陆运 0.3 方，开启后体积不足的按低消计费）
- 没配价格的客户应收按 0 计算

### 客户账期
- 查看每个客户的账单汇总（总额、已收、未收、最近付款）
- 显示客户评级（A/B/C/D）、平均回款天数、月均体积和票数
- 按客户名、唛头、月份筛选
- 录收入款（点击客户卡片上的对话框按钮）

---

## 6. 账单管理

### 生成账单
1. 点击"生成账单"，系统按唛头 + 月份自动生成账单
2. 账单金额按客户价格矩阵计算，低消规则自动生效
3. 生成后可查看每张账单的明细

### 刷新账单
- 如果客户价格变动，点击账单卡片上的"刷新"按钮
- 系统按最新客户价格矩阵重新计算金额
- 不会自动刷新，需手动触发

### 搜索账单
- 顶部搜索框支持按唛头、月份筛选

### 标记付款
- 在账单详情页标记付款状态（已付款/付一部分/待付款）
- 可上传付款凭证（水单）

### 导出账单
1. 点击"导出"，系统生成 Excel 文件
2. 包含模板头部（银行账号、二维码）、数据明细、合计行
3. 数据从数据库原样读取，不二次计算
4. 按运单号自动合并单元格

---

## 7. 支出总表

- 按费用类型查看所有支出汇总（CNY/THB）
- 自建费用明细（手动录入的费用）
- **拼柜成本**：按唛头分组，显示每条拼柜明细的费用，支持逐条支付和一键支付
- **装柜成本**：同拼柜成本逻辑
- 唛头卡片默认收起，点击展开箭头查看明细
- 支持上传凭证、编辑、删除

---

## 8. 收入总表（直接收入）

- 录入手动收入记录
- 新建/编辑/删除收入条目
- 关联唛头和客户

---

## 9. 月报

- 按月查看收入、支出、利润汇总
- 支持月份切换

---

## 10. 用户管理

- 管理员可创建/编辑用户
- 角色：管理员、财务、操作员、查看者

---

## 常用操作速查

| 操作 | 位置 |
|------|------|
| 上传拼柜 Excel | 拼柜管理 → 上传拼柜 |
| 审核拼柜数据 | 拼柜管理 → 点击批次 |
| 生成账单 | 账单管理 → 生成账单 |
| 导出账单 Excel | 账单管理 → 导出 |
| 刷新账单金额 | 账单管理 → 点击卡片刷新按钮 |
| 标记账单已付款 | 账单管理 → 点击账单 → 标记付款 |
| 客户设价格矩阵 | 客户管理 → 编辑客户 → 价格矩阵 |
| 某客户收款 | 客户账期 → 点卡片上的收款按钮 |
| 支付拼柜/装柜成本 | 支出总表 → 展开唛头 → 逐条支付或一键支付 |
| 查看月利润 | 月报 |
