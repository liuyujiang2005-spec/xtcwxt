## 2. 快速上手

### 2.1 登录

浏览器打开 `https://xtcwxt.site`，输入用户名和密码。

### 2.2 推荐操作顺序

1. **添加客户** → 设置价格矩阵
2. **添加供应商** → 标注类型
3. **新建票货** → 录入收入 + 成本

5. **查看报表** → 月度/账龄分析

## 3. 基础数据管理

### 3.1 客户管理 `/customers`

**新建客户**
- 客户名称（必填）
- 联系人（选填）
- 默认币种：CNY / THB
- 价格矩阵（6 个价格，单位：元/方）

| 运输方式 | 货物类型 | 价格字段 |
|----------|----------|----------|
| 海运 | 普货 | `sea_regular` |
| 海运 | 商检货 | `sea_inspection` |
| 海运 | 敏感货 | `sea_sensitive` |
| 陆运 | 普货 | `land_regular` |
| 陆运 | 商检货 | `land_inspection` |
| 陆运 | 敏感货 | `land_sensitive` |

> **提示**：价格矩阵被新建票货时自动引用，选客户 + 运输方式 + 货物类型后单价自动填入。

**编辑客户**
- 点击铅笔图标，修改信息或价格矩阵

**删除客户**
- 点击垃圾桶图标（仅管理员）

### 3.2 供应商管理 `/suppliers`

- 新建时填写名称和类型（车队/报关行/仓库/船司/其他）
- 操作权限同客户管理

## 4. 票货管理

### 4.1 票货列表 `/shipments`

- 表格显示：票号、客户、运输方式、体积、应收、成本、利润、状态
- **筛选条件**：月份 / 客户 / 状态 / 运输方式
- 点击眼睛图标查看详情

### 4.2 新建票货 `/shipments/new`

**第一步：填写基本信息**
1. 选择客户 → 自动带出价格矩阵
2. 选择运输方式（海运/陆运）+ 货物类型（普货/商检/敏感货）→ 自动填入单价
3. 填写体积（m³）→ 自动计算应收金额
4. 可选填写：提单号、柜号、ETD、ETA BKK、备注

**第二步：添加成本明细**
- 点击「添加成本」
- 每行选择：费用类型、金额（元）、币种、供应商
- 支持添加多行成本

> **费用类型**：国内拖车费、装柜费、报关费、海运费、清关费、THC、文件费、仓储费、派送费、杂费

**第三步：确认创建**
- 应收金额自动汇总，点击「创建票货」

### 4.3 票货详情 `/shipments/[id]`

**页面结构**：
- **基本信息卡片**：所有字段详情
- **利润汇总卡片**：应收 - 总成本 = 利润，已收/未收金额
- **成本明细表格**：每项费用明细，可删除（垃圾桶按钮）
- **回款记录表格**：该票被分摊到的所有回款

**操作**：
- **修改状态**：运输中 → 已到仓 → 已签收 → 已结算
- **删除成本**：点击成本行末尾的垃圾桶按钮（需确认）

### 4.4 票货状态流转

```
运输中 → 已到仓 → 已签收 → 已结算
                              ↓
                          部分已收（收款不足时）
```

## 5. 回款与付款

### 5.1 客户回款 `/accounts/customers`

**录入回款**
1. 点击「录入回款」
2. 选择客户
3. 输入金额（元）、选择币种、回款日期
4. 系统自动按 FIFO（先进先出）显示分摊预览
5. 点击「确认录入」

**FIFO 分摊规则**：
- 回款按该客户未结清票的创建时间从早到晚依次分摊
- 一部分回款可覆盖多票，一票可被多次回款

**页面功能**：
- 每个客户卡片显示：应收/已收/未收
- **账龄分析**：30天内、30-60天、60-90天、90天以上的未收金额
- **客户评分**：A/B/C/D 四级（基于回款天数、月货量、超期数）

### 5.2 供应商付款 `/accounts/suppliers`

**录入付款**
1. 点击「付款录入」
2. 选择供应商
3. **方式一（推荐）**：勾选关联费用，金额自动汇总
4. **方式二**：手动输入金额
5. 选择币种、付款日期，确认录入

## 6. 报表与分析

### 6.1 仪表盘 `/`

- **5 个汇总卡片**：本月营收、支出、利润、待收、待付（CNY + THB 分列）
- **近 6 个月趋势图**：营收 CNY/THB + 利润 CNY/THB（有 THB 数据时自动显示）
- **客户优质度 Top 5**：A/B/C/D 评级
- **近期回款记录**：最新 5 条回款

### 6.2 收入总表 `/revenue`

- 按客户汇总，CNY 和 THB 分列

### 6.3 支出总表 `/expenses`

- **按费用类型**：各类费用的 CNY/THB 金额
- **按供应商**：各供应商的 CNY/THB 金额

### 6.4 月度报表 `/reports/monthly`

1. 选择月份
2. 查看营收明细（按客户）、支出明细（按费用类型）
3. CNY/THB 分列显示
4. **导出 CSV**：点击「导出 CSV」下载报表

1. 点击「新建发票」

## 9. 常见问题

### Q1：为什么单价填了元但显示不一致？
系统内部以「分」存储（1 元 = 100 分），前端输入元自动转换为分提交，避免浮点精度问题。

### Q2：回款分摊预览的金额为什么和票货应收不一致？
回款按 FIFO 顺序分摊，先还早的票，预览显示的是算法自动分配的结果。

### Q3：多币种（CNY/THB）如何处理的？
所有汇总页面 CNY/THB 分列显示，不会直接相加。仪表盘卡片 CNY 主行 + THB 小字附注。

### Q4：如何导出数据？
月度报表页面 `/reports/monthly` 支持 CSV 导出。

### Q5：证书过期怎么办？
Let's Encrypt 证书 90 天自动续期，无需手动操作。
